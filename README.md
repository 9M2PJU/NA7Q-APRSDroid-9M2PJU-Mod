## About

**APRSdroid 9M2PJU Mod** is an Android client for the
[APRS (Automatic Packet Reporting System)](http://aprs.org/) network.
It is a fork of [NA7Q's enhanced APRSdroid](https://github.com/na7q/aprsdroid),
which is itself a fork of [Georg Lukas's original APRSdroid](https://aprsdroid.org/).

### Features

- Position reporting via GPS, fixed, periodic, and SmartBeaconing
- APRS messaging with conversations
- Offline maps (MapsForge + MBTiles) and online OSM tiles
- Bluetooth TNC (classic + BLE), USB serial TNC, DigiRig
- Digipeater and I-gate functionality
- AFSK audio modem
- APRS-IS connectivity (TCP/UDP/HTTP)
- Winlink APRSLink email gateway integration
- Mic-E compression and emergency status
- Material Design dark theme with bottom navigation
- In-app update checker

## Download

Download the latest signed release APK from the
[Releases page](https://github.com/9M2PJU/APRSdroid-9M2PJU-Mod/releases/latest)
or visit the [landing page](https://aprsdroid.hamradio.my/).

> **Note:** Uninstall any previous official version of APRSdroid before
> installing this mod - the signing key differs, so Android will refuse an
> in-place upgrade.
